About the Game

Carl's Sheety Situtaion is currently a game in progress. It is a simple game about an unfortunate front-end developer, Carl, who has lost all the CSS files. Help him find all the files to restore the website to normal.

The game came about as an exploration of the HTML5 canvas element and trying to come up with a clever game name using 'CSS' as an acronym. The game was built frantically over a course of around 3 or so weeks. Unintentionally the game has a slight educational spin to it and perhaps could be expanded into an entertaining way of teaching the basic concepts of CSS and maybe more.

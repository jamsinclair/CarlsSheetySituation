Uh oh, why does this website look so boring or broken? Oh it's Carl, he's lost all the CSS files (Cascading Style Sheets) for the website.

A CSS file is essentially a file with a list of rules on how a website should display. The browser takes those rules and applies it magically somehow to make a pretty(or ugly) looking website that we all know and love.

To help Carl keep his job and restore this website, navigate his desktop and place the files in the correct folder.

Controls:
Tap/Mouse: To move the cursor use your mouse or tap where you want it to move.
Double Tap/Click: To pick up/drop a file. Or interact with some folders and buttons
